# Kingdom Builder

Main Premise: Medieval Sims City BuildIt

  

Resource Collection to improve medieval plot
- Civ6 Land Bonuses? → Certain areas grant higher production value?

  

Buildings generate **Plot Satisfaction** amongst other values
- **High Satisfaction** boosts production and other factors in that area
- **Low Satisfaction** reduces production
-  When satisfaction is too low, peasants **Revolt**

Add Procedual Generated Ground:
- Check theres enough default space (looped)
- Add Trees
- Add Stone

Resource Depletion - Limited
- Player has to (eventually) go further out to collect
- Which hints outpost style gameplay (Roads & Carts)
